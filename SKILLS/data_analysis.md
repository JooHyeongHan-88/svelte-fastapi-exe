# 데이터 분석 가이드

## 절차
1. `add_todo` 로 아래 3단계를 한 번에 등록한다.
2. **수집** — 분석 대상 데이터의 출처와 범위를 명시한 뒤 `complete_todo`.
3. **정제** — 결측·이상치·중복을 식별하고 처리 방식을 1줄로 기록한 뒤 `complete_todo`.
4. **시각화 및 요약** — `display_chart` 로 분석 결과를 인터랙티브 차트(scatter/line/bar)로
   표시한 뒤, 핵심 인사이트 1~3가지를 수치 근거와 함께 제시하고 `complete_todo`.
   - 차트 유형은 데이터 특성에 맞게 선택한다 (상관관계 → scatter, 추세 → line, 비교 → bar).
   - 사용자는 차트에서 드래그 선택·확대·저장 도구를 직접 활용할 수 있다.

## 행동 원칙
- 각 단계마다 무엇을 했는지를 한 줄로 명확히 기록한다 (`complete_todo` 의 summary).
- 인사이트는 추론이 아닌 데이터로 뒷받침되어야 한다. 근거가 약하면 "잠정" 으로 표기.
- 분석 기간·세그먼트가 명확하지 않으면 추측 대신 보고에 "[기간 미지정]" 으로 명시.

## 금지
- 임의로 분석 대상을 확장하지 않는다 — 위임받은 범위만 다룬다.
- 통계 수치를 만들어 내거나 추정값을 단정형으로 보고하지 않는다.
