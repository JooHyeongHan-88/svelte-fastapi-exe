Backend Operation Flow

백엔드 동작 전체 흐름

EXE 더블클릭(기동)부터 채팅 한 턴이 처리·영속되기까지를 결정 게이트 · 병렬 분기 · 반복 루프로 한 장에 담았다. ◇ 조건 는 분기점, ▽ 병렬 은 동시 실행, 점선 박스는 루프다. 각 노드는 담당 모듈·함수를 표기하고 클릭하면 상세 흐름도로 간다.

① 정적 서빙 main.py ② HTTP API api/ ③ 에이전트 하니스 agent/harness ④ 생명주기 core/browser.py ⑤ 자동 업데이트 core/updater.py
