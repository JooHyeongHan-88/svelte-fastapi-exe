Sub-Agent Dispatch

⑥ 서브에이전트 디스패치 — 순차 · 병렬 · 격리

오케스트레이터가 복잡한 작업을 전문 서브에이전트에게 위임하는 경로. 순차(call_sub_agent)와 병렬(call_sub_agents_parallel) 두 방식을 게이트·fork/join·루프로 표현한다.
