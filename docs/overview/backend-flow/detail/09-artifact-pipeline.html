Artifact Pipeline

⑨ 산출물 파이프라인 — 세션을 넘는 데이터 영속

에이전트 작업 결과가 휘발되지 않고 다음 턴·다음 세션에서 재사용되는 구조. 저장 → manifest 기록 → 프롬프트 노출 → list/load 재사용의 순환.

1

저장 구조

result/ └─ <세션제목>-<id 앞8자>/ ← 세션 단위 ├─ _artifacts.jsonl ← 세션 manifest (산출물 장부) └─ <YYYYMMDD-HHmmss>/ ← 턴 단위 슬롯 (artifact_slot) ├─ data.parquet 데이터 (표준 포맷) ├─ charts.spec.json 차트 선언 ├─ charts.json 렌더된 ECharts option (파생물) └─ report.md 보고서
